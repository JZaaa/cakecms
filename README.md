# CakeCMS

基于CakePHP 3.7的内容管理系统

## 安装

```bash
git@github.com:JZaaa/cakecms.git
composer install
```

本项目使用MySQL数据库，请导入config/backupsql/xx/*.sql

框架初始化配置请参考[CakePHP 手册](https://book.cakephp.org/3.0/en/index.html)
